Program Start Guide:
1. Setelah extract maka akan ada folder "UAS_OOP" pada tempat dimana zip di extract
2. Masuk ke folder "UAS_OOP" dan run file "__init__.py"


Catatan:
1. Untuk tugas ini file yang digunakan hanya db.py, gui.py dan __init__.py
2. import UAS_OOP.oop.smarthome.db menandakan bahwa file db.py ada di dalam folder UAS_OOP/oop/smarthome/
3. Pastikan file .png dan .ico ada di dalam folder yang sama dengan __init__.py
4. Dalam program ini digunakan library :
    - tkinter
    - threading
    - pill/ pillow
    - time
    - configparser
    - sqlite3
5. File utama dalam proyek ini adalah
- gui.py
- services.py
- db.py
